# 一个包含vue和node的全栈项目

> A Vue.js project for vue and node

## 技术栈
* ### 前端技术
vue2 + vuex + vue-router + webpack + ES6/7 + axios + stylus + element-ui
* ### node技术
nodejs + express4 + session + sequelize +mysql
